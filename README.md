# Helena Me Manta - Manual de Identidad de Marca
Bienvenido al repositorio oficial del manual de marca de Helena Me Manta. Este proyecto documenta y preserva la esencia visual y narrativa de la marca, asegurando su correcta aplicación en cualquier entorno.

# Concepto y Filosofía
La identidad de Helena Me Manta nace de la intersección entre la mitología clásica y el diseño contemporáneo. El concepto central gira en torno a la figura de Helena de Troya, símbolo de belleza, fuerza y una presencia que deja huella.

# El Isotipo
El logotipo se fundamenta en una construcción geométrica donde la letra "h" minúscula se transforma sutilmente:

La inicial: Representa el nombre de la marca con claridad.

El cuerpo: El trazo vertical y la curva evocan la silueta de un cuerpo humano, aportando humanidad y cercanía.

Guiño Griego: Las proporciones y el estilo minimalista rinden homenaje a la estética de la Grecia clásica, conectando el pasado legendario con el presente digital.

# Contenidos del Manual
Este manual interactivo incluye las pautas esenciales para la marca:

Logometría: Construcción técnica y áreas de protección.

Paleta de Colores: Definición de colores principales y secundarios (HEX, RGB, CMYK).

Tipografía: Fuentes seleccionadas para mantener la jerarquía y legibilidad.

Uso de Imágenes: Guía de estilo visual y tono fotográfico.

Aplicaciones: Ejemplos de cómo vive la marca en diferentes soportes.

# Tecnologías Utilizadas
Este manual ha sido desarrollado como una experiencia web fluida y responsiva utilizando:

HTML5 / CSS3 (Custom Properties para la gestión de color).

JavaScript para interacciones dinámicas.
